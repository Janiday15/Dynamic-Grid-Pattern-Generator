# Dynamic Grid Pattern Generator

## Overview
A C++ console application that generates a customizable grid-based pattern using multi-dimensional arrays and repeating symbolic elements.

## Features
- User-defined grid size (rows and columns)
- Dynamic 2D vector structure
- Repeating pattern generation using modular indexing
- Styled grid border output

## Technologies
- C++
- Visual Studio Code

## Concepts Used
- Multi-dimensional arrays (vectors)
- Nested loops
- Modulo arithmetic for pattern generation
- Console I/O

## What I learned
- How to structure 2D data
- How to generate repeating patterns programmatically
- How to design clean console output systems
